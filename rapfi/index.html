每項執行 100 萬次、共 9 輪交錯執行並取中位數；全部在同一個 Web Worker 與同一台裝置測量。

Rapfi 搜尋在 Web Worker 中執行。單執行緒搜尋期間無法即時處理停止指令，因此「停止／重啟引擎」會終止 Worker 並重新載入。
